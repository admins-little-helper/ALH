
NAME
    Get-ALHADDomainController
    
SYNOPSIS
    Locate a domain controller.
    
    
SYNTAX
    Get-ALHADDomainController [[-DomainName] <String>] [[-Server] <String>] [[-DomainGuid] <Nullable`1>] [[-SiteName] <String>] [-DirectoryServiceRequired] [-DirectoryServicePreferred] [-GlobalCatalogRequired] [-PrimaryDCRequired] [-NoCache] [-UseCache] [-IpRequired] [-KdcRequired] [-TimeservRequired] [-WritableRequired] [-GoodTimeservPreferred] [-AvoidSelf] [-OnlyLdapNeeded] [-IsFlatName] [-IsDnsName] [-TryNextClosestSite] [-WebServiceRequired] [-Server2008OrLater] [-Server2012OrLater] [-Server2012R2OrLater] [-Server2016OrLater] [-ReturnDnsName] [-ReturnFlatName] [<CommonParameters>]
    
    
DESCRIPTION
    Locates a domain controller and returns the name and some additional information for about it.
