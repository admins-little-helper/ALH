
NAME
    Get-ALHSIDByAccount
    
SYNOPSIS
    Function to get an account's SID by account name and domain name
    
    
SYNTAX
    Get-ALHSIDByAccount [-Identity] <String[]> [[-Domain] <String>] [<CommonParameters>]
    
    
DESCRIPTION
    Function to get an account's SID by account name and domain name
    

PARAMETERS
    -Identity <String[]>
        Mandatory. One or more account names for which the SID shoudl be retrieved.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByValue)
        Accept wildcard characters?  false
        
    -Domain <String>
        Optional. The name of the domain to search accounts in.
        
        Required?                    false
        Position?                    2
        Default value                $env:USERDOMAIN
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
OUTPUTS
    
    -------------------------- EXAMPLE 1 --------------------------
    
    PS > Get SID for a single user in domain "domain.tld"
    Get-ALHSIDByAccount -Identity "User1" -Domain "domain.tld"
    
    
    
    
    
    
    -------------------------- EXAMPLE 2 --------------------------
    
    PS > Get SID for a multiple accounts in domain "domain.tld"
    Get-ALHSIDByAccount -Identity "User1","ComputerA$" -Domain "domain.tld"
    
    
    
    
    
    
    -------------------------- EXAMPLE 3 --------------------------
    
    PS > Get SID for a multiple accounts retrieved from a text file via pipeline input in domain "domain.tld"
    Get-Content -Path "C:\Temp\Accountnames.txt" | Get-ALHSIDByAccount -Domain "domain.tld"
