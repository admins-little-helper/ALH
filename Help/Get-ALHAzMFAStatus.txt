
NAME
    Get-ALHAzMFAStatus
    
SYNOPSIS
    Function to query MFA Status for a user account in Azure AD.
    
    
SYNTAX
    Get-ALHAzMFAStatus [[-UserPrincipalName] <String[]>] [<CommonParameters>]
    
    Get-ALHAzMFAStatus [[-AuthMethod] <String>] [[-IsDefault]] [<CommonParameters>]
    
    Get-ALHAzMFAStatus [[-State] <String>] [<CommonParameters>]
    
    
DESCRIPTION
    Function to query MFA Status for a user account in Azure AD.
    

PARAMETERS
    -UserPrincipalName <String[]>
        Mandatory. Username of the user to query MFA status for.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -AuthMethod <String>
        Using this parameter allows to filter the results for one of the following MFA Authentication methods:
        - PhoneAppNotification
        - PhoneAppOTP
        - TwoWayVoiceMobile
        - OneWaySMS
        - email
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -IsDefault [<SwitchParameter>]
        Use this parameter in combinaation with parameter AuthMethod to filter the results where the MFA authentication method is the default.
        
        Required?                    false
        Position?                    2
        Default value                False
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    -State <String>
        Using this parameter allows to filter the results for one of the following MFA Authentication states.
        - Disabled
        - Enabled
        - Enforced
        
        Note: When MFA is triggered through Conditional Access, the state is always Disabled.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       true (ByPropertyName)
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    
OUTPUTS
    
    -------------------------- EXAMPLE 1 --------------------------
    
    PS > Get-ALHAzMFAStatus
    This command retrieves all users within the domain
    
    
    
    
    
    
    -------------------------- EXAMPLE 2 --------------------------
    
    PS > Get-ALHAzMFAStatus -AuthMethod PhoneAppNotification
    This command retrieves all users  that have PhoneAppNotification enabled
    
    
    
    
    
    
    -------------------------- EXAMPLE 3 --------------------------
    
    PS > Get-ALHAzMFAStatus -UserPrincipalName alex@contoso.com
    This command lists the MFA configuration settings for the specified user.
    
    
    
    
    
    
    -------------------------- EXAMPLE 4 --------------------------
    
    PS > Get-ALHAzMFAStatus -State Disabled
    This command lists all users that have MFA explicitely enabled. if you use Conditional access and MFA
    you should not have any uses that have an MFA state of "Enabled" or "Enforced'
