
NAME
    ConvertFrom-ALHLocalTime
    
SYNOPSIS
    Contains function to convert a local date/time to miliseconds since 1970 or milliseconds since 1601 or ticks since 1601
    
    
SYNTAX
    ConvertFrom-ALHLocalTime [-DateTime] <DateTime> [<CommonParameters>]
    
    
DESCRIPTION
    Contains function to convert a local date/time to miliseconds since 1970 or milliseconds since 1601 or ticks since 1601
    

PARAMETERS
    -DateTime <DateTime>
        Date
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    DateTime
    
    
OUTPUTS
    PSCustomObject
    
    
NOTES
    
    
        Author:     Dieter Koch
        Email:      dieter.koch@sadadiko.net
    
    -------------------------- EXAMPLE 1 --------------------------
    
    PS > # Convert a int64 value to local date/time
    ConvertFrom-ALHLocalTime -DateTimeValue 13270022798437
    
    
    
    
    
    
    -------------------------- EXAMPLE 2 --------------------------
    
    PS > # Convert the value stored in lastLogonTimeStampe of a computer object in Active Directory to local date/time
    # This value is stored as a large integer that represents the number of 100-nanosecond intervals since January 1, 1601 (UTC).
    # https://docs.microsoft.com/en-us/windows/win32/adschema/a-lastlogontimestamp
    ConvertFrom-ALHLocalTime -DateTimeValue (Get-ADComputer -Identity MyComputer -Properties lastLogonTimestamp).lastLogonTimestamp
