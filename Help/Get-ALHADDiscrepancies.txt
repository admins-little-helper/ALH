
NAME
    Get-ALHADDiscrepancies
    
SYNOPSIS
    Function to run rules to identify discrepancies in Active Directory.
    
    
SYNTAX
    Get-ALHADDiscrepancies [[-ScriptSettingsFile] <String>] [[-RulesFile] <String[]>] [[-DomainName] <String>] [[-Server] <String>] [[-Credential] <PSCredential>] [-UseActiveDirectoryModule] [<CommonParameters>]
    
    
DESCRIPTION
    Function to run rules to identify discrepancies in Active Directory.
    

PARAMETERS
    -ScriptSettingsFile <String>
        Optional. String. Filepath to the script settings file in JSON format.
        
        Required?                    false
        Position?                    1
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -RulesFile <String[]>
        Optional. Array of strings. Filepath to the rules file in JSON format.
        
        Required?                    false
        Position?                    2
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -DomainName <String>
        
        Required?                    false
        Position?                    3
        Default value                (Get-CimInstance -ClassName "Win32_ComputerSystem").Domain
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Server <String>
        
        Required?                    false
        Position?                    4
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Credential <PSCredential>
        
        Required?                    false
        Position?                    5
        Default value                [System.Management.Automation.PSCredential]::Empty
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -UseActiveDirectoryModule [<SwitchParameter>]
        Optional. Switch. If specified, this function is using the ActiveDirecotry PowerShell module to query Active Directory objects.
        If omitted this function used the Get-ALHADObject function from the ALH module to query Active Directory.
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    None
    
    
OUTPUTS
    String
