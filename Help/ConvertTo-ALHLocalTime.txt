
NAME
    ConvertTo-ALHLocalTime
    
SYNOPSIS
    Converts a int64 value to multiple DateTime values
    
    
SYNTAX
    ConvertTo-ALHLocalTime [-DateTimeValue] <Double> [-AsLocalTime] [<CommonParameters>]
    
    
DESCRIPTION
    Converts a int64 value to multiple DateTime values
    

PARAMETERS
    -DateTimeValue <Double>
        Miliseconds since 01.01.1700 00:00:00 or Miliseconds since 01.01.1601 00:00:00 or ADDateTime value
        
        Required?                    true
        Position?                    1
        Default value                0
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -AsLocalTime [<SwitchParameter>]
        If specified, the resulting DateTime value will be interpreted in local time zone, instead of UTC
        
        Required?                    false
        Position?                    named
        Default value                False
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    Int64
    
    
OUTPUTS
    PSCustomObject
    
    
NOTES
    
    
        Author:     Dieter Koch
        Email:      dieter.koch@sadadiko.net
    
    -------------------------- EXAMPLE 1 --------------------------
    
    PS > # Convert a int64 value to local date/time
    ConvertTo-ALHLocalTime -DateTimeValue 13270022798437
    
    
    
    
    
    
    -------------------------- EXAMPLE 2 --------------------------
    
    PS > # Convert the value stored in lastLogonTimeStamp of a computer object in Active Directory to local date/time
    # This value is stored as a large integer that represents the number of 100-nanosecond intervals since January 1, 1601 (UTC).
    # https://docs.microsoft.com/en-us/windows/win32/adschema/a-lastlogontimestamp
    ConvertTo-ALHLocalTime -DateTimeValue (Get-ADComputer -Identity MyComputer -Properties lastLogonTimestamp).lastLogonTimestamp
