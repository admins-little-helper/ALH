
NAME
    Invoke-ALHTranslate
    
SYNOPSIS
    Function to translate text using the DeepL API.
    
    
SYNTAX
    Invoke-ALHTranslate [-TextToTranslate] <String> [-TargetLanguage] <String> [[-SourceLanguage] <String>] [[-SplitSentences] <String>] [[-PreserveFormatting] <String>] [[-Formality] <String>] [[-GlossaryId] <String>] [-ApiKey] <String> [<CommonParameters>]
    
    
DESCRIPTION
    Function to translate text using the DeepL API. To use the DeepL API, either a free or pro account is required.
    

PARAMETERS
    -TextToTranslate <String>
        Mandatory. String. Text to be translated. Only UTF8-encoded plain text is supported.
        
        Required?                    true
        Position?                    1
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -TargetLanguage <String>
        Mandatory. String. The language into which the text should be translated
        
        Required?                    true
        Position?                    2
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -SourceLanguage <String>
        Optional. String. Language of the text to be translated
        
        Required?                    false
        Position?                    3
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -SplitSentences <String>
        Optional. String. Sets whether the translation engine should first split the input into sentences. This is enabled by default.
        
        Required?                    false
        Position?                    4
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -PreserveFormatting <String>
        Optional. String. Sets whether the translation engine should respect the original formatting, even if it would usually correct some aspects.
        
        Required?                    false
        Position?                    5
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -Formality <String>
        Optional. String. Sets whether the translated text should lean towards formal or informal language.
        
        Required?                    false
        Position?                    6
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -GlossaryId <String>
        Optional. String. Specify the glossary to use for the translation.
        Important: This requires the source_SourceLanguagelang parameter to be set and the language pair of the glossary has to match the language pair of the request.
        
        Required?                    false
        Position?                    7
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    -ApiKey <String>
        Optional. String. You need an authentication key to access to the DeepL API.
        Refer to the DeepL API Documentation for more information.
        
        Required?                    true
        Position?                    8
        Default value                
        Accept pipeline input?       false
        Accept wildcard characters?  false
        
    <CommonParameters>
        This cmdlet supports the common parameters: Verbose, Debug,
        ErrorAction, ErrorVariable, WarningAction, WarningVariable,
        OutBuffer, PipelineVariable, and OutVariable. For more information, see
        about_CommonParameters (https://go.microsoft.com/fwlink/?LinkID=113216). 
    
INPUTS
    String
    
    
OUTPUTS
    PSCustomObject
    
    
NOTES

    Invoke-ALHTranslate -TextToTranslate "PowerShell is cool!" -SourceLanguage "EN" -TargetLanguage "DE" -ApiKey "<YourApiKey>"
